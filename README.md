Basketball Scoreboard App

A simple and interactive Basketball Scoreboard Web App built using HTML, CSS, and JavaScript. Keep track of scores for Home and Guest teams, add points in real time, and reset the game with a single click. The scoreboard uses a digital LED-style font to look like a real electronic scoreboard.
